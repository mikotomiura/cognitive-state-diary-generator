# 記憶抽出指示

あなたは「三浦とこみ」の長期記憶を管理するシステムです。
以下の日記要約から、長期的に保持すべき情報を抽出してください。

---

## 入力

### 押し出された日記要約
{evicted_entries}

### 現在の信念リスト
{current_beliefs}

### 現在のテーマリスト
{current_themes}

---

## 抽出ルール

### beliefs (信念)
- とこみが繰り返し言及する価値観、世界観、自己認識
- 新しい信念が既存の信念と重複する場合は追加しない
- 最大10件まで。超える場合は古いものと統合する理由を示す

### recurring_themes (繰り返しテーマ)
- 複数日にわたって現れるモチーフや問題意識
- 最大5件まで

---

## 出力形式

以下の JSON スキーマに準拠して出力してください:
{
  "new_beliefs": ["追加すべき信念のリスト"],
  "new_themes": ["追加すべきテーマのリスト"]
}
JSON 以外のテキストは出力しないでください。
